# Glossary

> **Terms and definitions**

## General Terms

**Jarvis**: The H&H AI Assistant platform name

**RAG**: Retrieval-Augmented Generation - technique combining search with AI generation

**Thread**: A conversation session with persistent history

**Tool**: A function that agents can invoke to perform tasks

## Technical Terms

**Agent**: Autonomous AI entity that uses tools to accomplish tasks

**Embedding**: High-dimensional vector representation of text (768-dim)

**Vector Database**: Database optimized for similarity search (ChromaDB)

**Semantic Search**: Search based on meaning rather than keywords

**Context Window**: Amount of conversation history the AI can remember

**Streaming Response**: Response delivered in real-time as it's generated

**Tool Calling**: Agent's ability to invoke external functions

## Framework Terms

**Mastra**: Agent orchestration framework

**assistant-ui**: React components for chat interfaces

**Ollama**: Local LLM runtime

**ChromaDB**: Vector database

**Supabase**: Backend-as-a-service (PostgreSQL + Auth)
